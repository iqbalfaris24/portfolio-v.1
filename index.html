Tentang Saya

          Saya adalah mahasiswa di Universitas Bhayangkara di fakultas Ilmu Komputer. Saya sudah memulai kuliah di tahun 2020 dan sekarang saya sudah masuk semester 2, saya memilih fakultas ini karena saya suka dunia komputer terutama bagian web development. Saya pernah magang di perusahaan PT Intraco Penta pada tahun 2018 di bagian admin selama 3 bulan, saya mendapat nilai yang memuaskan dari perusahaan, saya memiliki ambisi yang tinggi. Saya benar-benar dapat bekerja dengan orang-orang di sekitar saya untuk saling memberikan ide dan kreativitas sehingga saya dapat membuat sesuatu yang hebat. Saya berharap menjadi manusia yang bertanggung jawab dan menjadi yang terbaik. Saya juga suka mengikuti perkembangan teknologi sehingga saya bisa lebih kreatif dan inovatif.
